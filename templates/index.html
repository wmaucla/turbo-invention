Welcome to Turbo Invention!

An End to End Data Science Project

What is Data Science?

Data Science has been considered one of the hot new jobs of the century, but many companies are misinformed about what data scientists can/should do. This is my attempt to create a project - end to end - that showcases the power of data scientists. It involves computer vision, natural language processing, severless deployment, and much, much more! None of these technologies are revolutionary on their own, but knitted together they can truly help companies leverage AI.

Designed with Flask & Bootstrap

Data scientists should have the ability to create simple, elegant frontends for projects. Here I'm using flask to create a simple frontend based on a free bootstrap theme (see here).

Transformers (NLP)

NLP is a hot research topic. Pretrained models have fundamentally changed the game - now, we can leverage state of the art models without having to train from scratch! Here, I'm using the fantasic hugging face transformers library's pretrained BERT model for question answering. Check it out.

Severless Deployment

The difference between a data scientist and a data engineer depends on who you ask. To me, a data scientist needs to be able to handle simple deployments. Whether it's configuring build pipelines for CI/CD or handling containerization, these are all useful skills to have. For this project, I've chosen to have it deployed on AWS Lambda via a great framework called Zappa. This is truly an end-to-end application!
