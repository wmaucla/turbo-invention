Question Answer

What is Question Answer?

Question Answer is a NLP technique in which we try and build a model that allows us to ask a question and obtain a response from the text itself. The following comes from huggingface transformer's website:

Give the following text,
Extractive Question Answering is the task of extracting an answer from a text given a question. An example of a question answering dataset is the SQuAD dataset, which is entirely based on that task. If you would like to fine-tune a model on a SQuAD task, you may leverage the `run_squad.py`.

The question is "What is extractive question answering?"
The response we would expect would be "the task of extracting an answer from a text given a question"
